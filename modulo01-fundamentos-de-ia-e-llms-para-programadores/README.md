# Módulo 01 — Fundamentos de IA e LLMs para Programadores

Introdução prática a redes neurais e sistemas de aprendizado de máquina aplicados no browser e em Node.js com TensorFlow.js. Os projetos deste módulo cobrem desde a construção de uma rede neural simples até um sistema de recomendação completo com arquitetura MVC.

## Projetos

| # | Projeto | Descrição |
|---|---------|-----------|
| 01 | [neural-network](./01-neural-network/) | Rede neural que classifica usuários em categorias com TensorFlow.js |
| 02 | [Sistemas-Recomendacao](./02-Sistemas-Recomendacao/) | Sistema de recomendação de produtos para e-commerce com Web Worker e MVC |

## Conceitos abordados

- Arquitetura de redes neurais (camadas densas, ativações ReLU e Softmax/Sigmoid)
- Codificação de features: normalização numérica e one-hot encoding
- Treinamento e inferência com TensorFlow.js
- Execução de modelos em Web Worker para não bloquear a UI
- Arquitetura MVC com barramento de eventos (pub/sub) no frontend
- Binary Cross Entropy e Categorical Cross Entropy como funções de perda
- Otimizador Adam e seus hiperparâmetros
